# 表情包生成器
通过调用API生成表情包


## 使用
`/[命令] <@user> `  
> `[ ]`为必选参数,`< >`为可选参数  
> 如果不填写`< >`则生成使用命令的

### 参数
`Key`请前往`https://api.xiaoyu17love.top/` 申请

### 指令
| 命令    | 描述          | 默认权限 |
|:------|:------------|:---|
| doro  | 生成一个doro砸脑袋 | 普通 |
| arona | 生成一个阿罗娜丢出去  | 普通 |

## 平台支持
- QQ
