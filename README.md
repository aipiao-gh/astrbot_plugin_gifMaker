# 表情包生成器
通过调用API生成表情包


## 使用
`/[指令] <@user> `  
> `[ ]`为必选参数,`< >`为可选参数  
> 如果不填写`< >`内的内容则生成自己的

### 参数
`Key`请前往`https://api.xiaoyu17love.top/` 申请

### 指令
| 命令    | 描述          | 权限 |
|:------|:------------|:---|
| doro  | 生成一个doro砸脑袋 | 普通 |
| arona | 生成一个阿罗娜丢出去  | 普通 |
